# MD Editor — Feature Overview

## 🖊️ Split-Pane Editing

Write in Markdown on the left, see rendered HTML on the right — **in real time**. Scroll sync keeps both panes aligned as you type.

## 📑 Multiple Tabs

Work on several documents simultaneously with a familiar tab interface. Each tab remembers its view mode independently.

## 🎨 Themes

Choose the look that works for you:

| Theme               | Best for                            |
|---------------------|-------------------------------------|
| **Light**           | Bright environments, printing       |
| **Dark**            | Night coding sessions               |
| **Solarized Light** | Reduced eye strain, long sessions   |
| **Solarized Dark**  | Low-light, high contrast            |

## ⚡ Keyboard-First

Every action has a keyboard shortcut. Common ones:

- `⌘N` — New file in new tab
- `⌘O` — Open file
- `⌘S` — Save
- `⌘W` — Close tab
- `⌘⇧P` — Toggle preview
- `⌘F` — Find & replace

## 📤 Export

Export your documents to:
- **HTML** — Full standalone page with styles
- **PDF** — Via system print dialog

## 🔍 Code Highlighting

All major programming languages are supported:

```rust
fn main() {
    println!("Hello from MD Editor!");
}
```
