# Welcome to MD Editor

> A minimal, distraction-free Markdown editor for macOS, Windows, and Linux.

MD Editor combines a powerful **CodeMirror 6** editor with a live preview pane, giving you the best of both worlds — rich editing with instant visual feedback.

---

## ✨ Features

- [x] Live split-pane preview
- [x] Syntax highlighting for 100+ languages in code blocks
- [x] Multiple tabs for working on several files at once
- [x] Dark mode, Solarized Light/Dark, and Light themes
- [x] Export to HTML and PDF
- [x] Auto-save support
- [x] Scroll sync between editor and preview
- [x] File drag & drop
- [x] Recent files menu

## 📊 Keyboard Shortcuts

| Action         | macOS        | Windows/Linux   |
|----------------|--------------|-----------------|
| New File       | ⌘ N          | Ctrl N          |
| Open File      | ⌘ O          | Ctrl O          |
| Save           | ⌘ S          | Ctrl S          |
| Find           | ⌘ F          | Ctrl F          |
| New Tab        | ⌘ ⇧ N        | Ctrl Shift N    |
| Close Tab      | ⌘ W          | Ctrl W          |
| Toggle Preview | ⌘ ⇧ P        | Ctrl Shift P    |
| Preferences    | ⌘ ,          | Ctrl ,          |

## 💡 Tips

> **Tip:** Use the toolbar buttons to quickly insert formatting, or learn the keyboard shortcuts for a faster workflow.

You can drag and drop `.md` files directly onto the editor window to open them.
